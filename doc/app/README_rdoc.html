    README

    これは、Codestと言い、卒業制作として作ったWebアプリケーションです。

    このアプリケーションは、プログラミングを学びたい 作者の様な未熟なプログラマのために作られた、 Codest と言い、プログラミングの学習Webアプリケーションです。

    これを使う事で、コードを読む力と書く力が付き、 他の学習システムとは違い、問題とそれに付属するテストコードが付いてます。

    Gitからのクローン方法

    • git clone https://github.com/smntkm/Codest.git

    バージョン

    • Ruby 2.0.0

    • Rails 4.0.0

    • PostgrSQL 9.2.4

    gemのインストール

    • 依存gemのインストール bundle install.

    データベースの設定

    • データベースの作成 rake db:create.

    • スキーマの設定 rake db:migrate.

    • テストデータベースの初期化 rake db:testprepare.

    アプリの起動

    • アプリの起動 rails s.

    テストの実行

    • テスト実行 rake spec.

    APIドキュメント

    • APIドキュメントの作成 rake doc:app

    • APIドキュメントの観覧は、Codest/doc/app/index.html
